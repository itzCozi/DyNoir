DyNoir Usage Guide

DyNoir is a exploit not a program or website this means to use the exploit you must follow all steps below.

Step-by-Step Guide

1. Download the 'index.html' called base page here.
2. Install the 'Coding With Chrome' extension.
3. Press the 'Search-key' and select 'Coding With Chrome'.
4. Once open press 'USE ADVANCE MODE' option if prompted.
5. Now click the 'markup' option and 'USE HTML5' option.
6. After click the 'HTML5 Project' option.
7. Once a project is made paste the base file to it.
8. Now any links opened in 'CWC' will be invisible to Dyno users.
